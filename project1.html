    • HR Payroll System - PHP Developer - Laravel

    A detailed showcase of my design and development work

    Project Overview

    It is a spreadsheet system designed to completely transform the operational efficiency of the company. Unlike manual methods, the system automates processes, eliminating repetitive and error-prone tasks, allowing the team to focus on higher-value activities.

    This system is specially designed for companies that offer accounting services and is completely multi-company, allowing you to efficiently manage the spreadsheets of multiple clients from a single platform. The software consists of the development of a Comprehensive Payroll System that integrates with the PostgreSQL-based accounting system and the existing vacation system. Figma was used to create the system proposal.


    Company Overview: Outsourcing company specializing in software development.
    • Developed a payroll system using PHP and the Laravel framework, incorporating the Moonshine library.
    • Applied the MVC architecture by managing routes, controllers, models, and resources.
    • Performed basic operations with PostgreSQL databases, including queries and modifications via phpMyAdmin.
    • Specialized in FrontEnd development and project structuring while collaborating with team members using Scrum methodology for effective communication and project development.
